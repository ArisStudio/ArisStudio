Welcome to Aris Studio Installer

First, thank you for using Aris Studio, Sensei! We glad you give it a try.
This installer will install Aris Studio at your desired folder and
optionally, create a desktop shortcut to easily run Aris Studio.

To know more how to use Aris Studio, read the documentation at:
https://as.t14.me/en/docs/category/as-commands/

If you interested in Aris Studio development and want to contribute, checkout our repo at:
https://github.com/Tualin14/ArisStudio
